  • Here's some more info:

  • What's this all about?
    • This is ideal for quickly sharing a file among local groups, such as in a conference or office environment. The human-readable url makes it trivial to speak the link so a person or group can quickly pull it up and get the file. Because you don't need to log in to access the files, it shields your personal details on public computers, such as when you need to print a document from a public machine. Simply use your computer or smartphone to upload the document so you can easily pull it up on a public computer, and then self-destruct the link when you're done. No need to email yourself the document and log into your personal account on a public computer, or to upload to a another service and fumble trying to type in the document download url correctly.
  • Is there a restriction on file type?
    • Unregistered users are restricted to mostly document-type file extensions and smaller files. No illegal or copyrighted items. This isn't meant to be a music or video sharing service, so those file types are generally restricted.
  • Is there a file size limit?
    • The file limit is 20 megabytes, which is enough for the vast majority of files for which this service was intended. Pro users have a much higher limit.
  • Is my data secure?
    • The data is sent via SSL directly to Amazon S3 servers where it is encrypted and stored. However, although there are nearly a billion possible auto-generated urls for the download link and they generally only last a short time before self-destructing, anyone with the download link can download the file. Pro users do get password protection on the download links, however. No credit card information touches the servers and it sent directly to Stripe with their Stripe.js module. However, the file is completely deleted from the servers after it expires. Once it's gone, it's gone!
